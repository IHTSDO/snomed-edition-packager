# Snomed Edition Packager

Snomed Edition Packager is a command line tool providing the ability to combine several RF2 packages into one.

## Getting started

### Software requirements

- Java 17
- Maven 3.9+

### Installation

- Clone the repository
- Build the project
    - `mvn clean package`

### Running

- Run via Java
    - `java -Xms14G -Xmx14G -jar target/snomed-edition-packager-executable.jar`
- Once started, the application will be waiting for input commands

## Documentation

All other documentation, including the usage guide, can be found in the `docs` folder.
